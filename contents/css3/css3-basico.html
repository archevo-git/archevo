CSS3

BÁSICO

CSS: Abreviação de Cascading Style Sheets ou Folhas de Estilo em Cascata.

Descrição: Linguagem de estilo usada para descrever como os elementos de um documento HTML ou XML (incluindo várias linguagens em XML como SVG, MathML ou XHTML) será apresentado na tela, no papel, na fala ou em outras mídias.

Criação: 1994, por Hakon Lie

Extensão do Arquivo: .css

MIME: text/css

Tipo de Formato: Folha de estilo

CSS Inline, Embedded/Internal ou External
  • Inline: Estilo definido no próprio elemento da página HTML, usando o atributo style. É útil quando queremos estilizar um único elemento;
  • Embedded (Internal): Estilo definido com a tag <style>, no <head> da página HTML. É útil quando queremos estilizar uma única página;
  • External: Todas as regras de estilo são salvas em um arquivo .css e, no <head>, usamos a tag <link> referenciar esse arquivo;
