O que é o CSS3?

  • CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou em XML (incluindo várias linguagens em XML como SVG, MathML ou XHTML). O CSS descreve como elementos são mostrados na tela, no papel, na fala ou em outras mídias;
  • Cascading refere-se ao jeito que o CSS aplica um estilo em cima de outro;
  • Style Sheets controla a aparência dos documentos web;
  • O principal benefício do CSS é que ele permite separar o estilo do conteúdo;

Box Model

  • Os elementos em uma página web são caixas retangulares e contém 4 camadas: conteúdo, preenchimentos (paddings), bordas e margens:
  • padding: Gera um espaço entre o conteúdo e a borda;
  • margin: Gera um espaço envolta do elemento, entre a borda e os outros conteúdos;
  • Sequência para configurar os valores em uma linha do padding e da margin: top, right, bottom, left.
