Sobre o Python

    - É fácil de aprender, é de fácil manutenção e possui o modo interativo;
    - É orientado à objetos, multi-propósito e uma linguagem de alto nível;
    - Está presente nas áreas de Desenvolvimento Web, IA, Computação Gráfica, Big Data, Ciência de Dados, etc.;
    - É case sensitive;
