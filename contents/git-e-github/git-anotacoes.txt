GIT 

O que é o Git?

 - Git é um software de controle de versão que salva a trajetória das mudanças do seu projeto e, em casos de erros, 
 você pode voltar para versão funcional;
 - Esse controle de versão também possibilita que vários desenvolvedores trabalhem juntos em um mesmo projeto;
